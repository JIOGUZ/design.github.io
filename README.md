# VITRINA GRÁFICA Y MULTIMEDIA - JIO GUZ  

Este proyecto es una página web de muestra que presenta un portafolio de diseños gráficos y videos creativos realizados con herramientas como Adobe y Magix.  

## 🖼️ Descripción  
La página muestra una galería de diseños gráficos creados con Adobe y una selección de videos creativos alojados en YouTube. Está diseñada para ofrecer una experiencia visual impactante y accesible desde cualquier dispositivo.  

## 📋 Estructura del Proyecto  
### HTML  
El archivo `index.html` contiene:  
- **Encabezado:** Título principal y descripción de la página.  
- **Galería de Diseños Gráficos:** Muestra imágenes de trabajos realizados con Adobe Photoshop.  
- **Videos Creativos:** Presenta videos de YouTube con temas variados como consejos, recorridos y retrospectivas.  
- **Pie de página:** Información de contacto y derechos reservados.  

### CSS  
El archivo `sheetstyle.css` incluye los estilos para:  
- Un diseño limpio y moderno con colores contrastantes.  
- Galería de imágenes con efecto de sombra y escalado al pasar el cursor.  
- Contenedores de videos ajustados para una visualización óptima en diferentes dispositivos.  

## 🚀 Características  
1. **Diseños Gráficos:**  
   - Diseños realizados con herramientas como Adobe Photoshop.  
   - Imágenes alojadas localmente en la carpeta `IMG`.  

2. **Videos Creativos:**  
   - Enlaces a videos de YouTube insertados con `iframe`.  
   - Títulos y descripciones explicativas para cada video.
